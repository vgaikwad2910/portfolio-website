# Portfolio Website

My personal portfolio website showcasing data science projects, blog posts, and career journey.

**[Live Site →](https://vgaikwad2910.github.io/portfolio-website/)**

## Features

- Responsive design with dark/light mode toggle
- Interactive project demos embedded directly in the site
- Career timeline and skills visualization
- Blog section for technical writing
- Contact form

## Projects Featured

- [Netflix Recommendation Engine](https://github.com/vgaikwad2910/netflix-recommendation-engine)
- [A/B Testing Simulator](https://github.com/vgaikwad2910/ab-testing-simulator)
- [GenAI Support Analyzer](https://github.com/vgaikwad2910/genai-support-analyzer)
- [Supply Chain Dashboard](https://github.com/vgaikwad2910/supply-chain-dashboard)

## Tech Stack

HTML, CSS, JavaScript — no frameworks, no build step. Hosted on GitHub Pages.

## Run Locally

```bash
git clone https://github.com/vgaikwad2910/portfolio-website.git
cd portfolio-website
open index.html
```
